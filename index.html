Final Project Proposal

Project Name: EZMessage

Team Members: Adrian, Paula, Steve, Diya, Mariela

Last Update: 2021-12-04

Application Overview:

An application that will allow a user to be able to log in and out,
view when other users are online or offline, and send and receive
messages that will show a time stamp of when it was sent.
Messages that a user will be able to send will include emojis.
A user will be able to create set times to send premade messages
and a user will be able to delete previously sent messages.

Technical Overview

  • Languages: for this project the main languages is JavaScript, it is
    being used to to had the node server with express
    HTML and CSS is being used to format the webpage.
  • Libraries/modules: node_modules are being used along with express
    to handle the web routes, socket.io-client is being user to talk to the
    socket server to texting, other things like the google firebase and the emoji
    have links in the html to get files over the CDN
  • Hosting of the main page and the socket.io-server is being handled by Azure which is a Microsoft cloud hosting
    provider, this project created a web app in Aruze and was deployed
    using github actions
